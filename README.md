# Obsidian-卡片盒-启动套件
一个用于 **Obsidian** 的 **启动套件**，包含构建您自己的 **卡片盒** 系统的所有基本要素。

## Obsidian
Obsidian 是由 Shida Li 和 Erica Xu 开发的笔记和知识管理应用程序。它基于本地纯文本 Markdown 文件夹运行，让您能够将一系列纯文本文件转换为丰富的关联思维网络。

了解更多：https://obsidian.md

## 卡片盒（Zettelkasten）
卡片盒是由 Niklas Luhmann 创建的个人思考和写作工具。它具有超文本特性，使思维网络成为可能。与其他系统的不同之处在于，您创建的是思维网络而不是任意大小和形式的笔记，并强调连接而不是收集。

了解更多：https://www.soenkeahrens.de/en/takesmartnotes

## 使用方法
- 下载 zip 文件并解压
- 在 Obsidian 中选择解压后的文件夹作为笔记库
- 安装 Obsidian 社区插件：[Banners](https://github.com/noatpad/obsidian-banners)、[Charts](https://github.com/phibr0/obsidian-charts) 和 [DataView](https://github.com/blacksmithgu/obsidian-dataview)
- 阅读 [Sönke Ahrens](https://www.soenkeahrens.de/en/takesmartnotes) 的书籍
- 探索启动套件
- 学习使用 [Obsidian-卡片盒模板](https://github.com/groepl/Obsidian-Templates?tab=readme-ov-file#list-of-templates)
- 开始使用！

## 卡片盒定义
卡片盒可以定义为一个个人知识管理系统，由五个关键组件组成：笔记、链接、元数据、结构和流程。在研究、学习和写作的背景下，卡片盒旨在支持想法生成、知识发展和长期学习。理解卡片盒基本概念的最佳方式是查看一个典型的示例（原型或启动套件）。

但卡片盒并不是一个具有固定边界的单一事物。它是一系列在目的、结构和功能上相似的方法。您不需要所有特征——您只需要足够的特征来归属其中。
